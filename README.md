📊 Customer Retention & Churn Analysis
🔍 Project Overview
This project analyzes customer data to understand churn behavior and retention patterns in a subscription-based business. The objective is to identify why customers leave and suggest strategies to improve retention.

🎯 Objectives
Calculate overall churn rate

Identify high-risk customers

Analyze customer lifetime (tenure)

Understand key factors affecting churn

Provide data-driven recommendations

🛠️ Tools Used
Microsoft Excel

Data Cleaning

Pivot Tables

Dashboard Creation

📁 Dataset
The dataset contains customer details such as:

Demographics

Subscription type

Monthly & total charges

Contract type

Churn status (Yes/No)

📊 Analysis Performed
✅ Churn Rate
Calculated the percentage of customers who stopped using the service.

✅ Segmentation Analysis
Analyzed churn based on:

Contract type

Internet service

Payment method

✅ Customer Lifetime Analysis
Grouped customers based on tenure to study retention trends.

✅ High-Risk Customers
Identified customers likely to churn based on behavior patterns.

📈 Dashboard Features
KPI Cards (Total Customers, Churn Rate)

Churn by Contract Type

Churn by Services

Customer Tenure Distribution

🔍 Key Insights
High churn in month-to-month contracts

Most customers leave within the first year

Higher charges increase churn risk

Lack of additional services leads to higher churn
